## 🔧 Key Features

### Motor Control
- **6x DC Motor Drivers** (BTS7960) - High-current motor control
- **4x Encoder Inputs** - Quadrature encoding for closed-loop control
- **4x Servo PWM Outputs** - For servo motors and actuators

### Communication
- **6x UART Ports** - Serial communication 
- **2x I2C Buses** - Sensor integration 
- **CAN Bus** - Inter-board communication
- **Ethernet**  (LAN8270) - Network connectivity

### Digital I/O
- **5x Digital Inputs** - Button/switch inputs
- **3x Digital Outputs** - Control external devices
- **2x LED Indicators** - Status display (included in outputs)

### Power
- **12V-24V Input** - Battery powered
- **5V Regulated Output** - Logic power
